# Contributing

Thanks for helping improve Balik Alindog Tracker.

1. Create a focused branch from `main`.
2. Install dependencies with `npm install`.
3. Make the smallest coherent change and add or update tests.
4. Run `npm run lint`, `npm test`, and `npm run build`.
5. Open a pull request using the repository template.

Do not weaken the append-only measurement rules without documenting the product decision and migration impact. Never commit personal measurement exports, API keys, `.env` files, or other secrets.
